# Interactive Resume Website

An interactive resume website developed using **HTML, CSS, and JavaScript** as part of the Full Stack Development Lab.

## Features
- Responsive Design
- Navigation Bar
- Education & Projects
- Skills Section
- Contact Form
- Dark/Light Mode
- Form Validation
- Dynamic JavaScript Features

## Technologies Used
- HTML5
- CSS3
- JavaScript
